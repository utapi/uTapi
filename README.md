# uTapi

This app is graduation work at Nippon Engineering College.
https://u-tapi.web.app/

## Install

```
npm install -g ionic
git clone https://github.com/utapi/uTapi.git
cd uTapi
npm install
ionic serve
```

## Building
```
ionic build
```

## Deploy
```
firebase deploy
```
